# Rails Sales taxes

> Simple rails app where users upload a file with the content of a basket then prints the receipts on the web page.


## Built With

- Ruby v2.7.1
- Ruby on Rails v6.0.4

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

Ruby: 2.7.1

npm: run <code> npm install </code>

yarn

Rails: 6.0.4


### Setup

Clone the repository with:
```
git clone https://github.com/safafa/rails-sales-taxes-kata.git
```
Move to the root directory

Instal gems with:

```
bundle install
```

Setup database with:

```
   rails db:create
   rails db:migrate
```



### Usage

Start server with:

```
    rails server
```

Open `http://localhost:3000/` in your browser.

### Run tests

```
    rspec 
```
